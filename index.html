Instruções

Utilizando a API Olinda do Banco Central do Brasil (https://olinda.bcb.gov.br/olinda/servico/PTAX/versao/v1/aplicacao#!/recursos/CotacaoDolarDia), desenvolva uma aplicação que receba um valor em reais do usuário e exiba quantos dólares ele consegue comprar com o valor informado.

Para tal, a aplicação deve realizar a consulta tendo como referência a data atual.

Observe a documentação da API para informar corretamente o parâmetro de data e conseguir realizar a consulta.
